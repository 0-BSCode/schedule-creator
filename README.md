
# Schedule Creator

A schedule creator for USC students

## Run Locally

Clone the project

```bash
  git clone https://github.com/0-BSCode/schedule-creator
```

Go to the project directory

```bash
  cd schedule-creator
```

Install dependencies

```bash
  npm run setup
```

Start the server

```bash
  cd server
  npm run dev
```

Start the client

```bash
  cd client
  npm run dev
```
